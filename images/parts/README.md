# Изображения запчастей

## Как добавить изображения

1. Поместите изображения в эту директорию (`images/parts/`)
2. Назовите файлы точно так, как указано в базе данных (поле `image` в таблице `parts`)

## Текущие названия файлов в базе данных:

- `brake-pads.jpg` - Тормозные колодки передние
- `oil-filter.jpg` - Масляный фильтр
- `air-filter.jpg` - Воздушный фильтр
- `battery.jpg` - Аккумулятор 60Ah
- `spark-plugs.jpg` - Свечи зажигания
- `shock-absorber.jpg` - Амортизатор передний
- `timing-belt.jpg` - Ремень ГРМ
- `brake-disc.jpg` - Тормозной диск
- `starter.jpg` - Стартер
- `stabilizer-link.jpg` - Стойка стабилизатора

## Требования к изображениям:

- **Формат**: JPG, PNG или WebP
- **Рекомендуемый размер**: 800x600px или 1200x900px
- **Соотношение сторон**: 4:3 (желательно)
- **Размер файла**: до 500KB (для быстрой загрузки)

## Если нужно изменить название изображения:

Обновите поле `image` в базе данных:

```sql
UPDATE parts SET image = 'новое-название.jpg' WHERE id = 1;
```

## Пример добавления изображения:

1. Найдите изображение запчасти
2. Переименуйте его, например: `brake-pads.jpg`
3. Поместите в директорию `images/parts/`
4. Обновите страницу - изображение появится автоматически!

## Placeholder
