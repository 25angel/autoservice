# Изображения

## Структура

- `parts/` - изображения запчастей

## Как добавить изображения запчастей

1. Поместите изображения в директорию `images/parts/`
2. Назовите файлы по имени из базы данных (поле `image` в таблице `parts`)
3. Поддерживаемые форматы: JPG, PNG, WebP
4. Рекомендуемый размер: 800x600px или 1200x900px

## Примеры названий файлов

Согласно базе данных, изображения должны называться:
- `brake-pads.jpg` - для тормозных колодок
- `oil-filter.jpg` - для масляного фильтра
- `air-filter.jpg` - для воздушного фильтра
- `battery.jpg` - для аккумулятора
- `spark-plugs.jpg` - для свечей зажигания
- `shock-absorber.jpg` - для амортизатора
- `timing-belt.jpg` - для ремня ГРМ
- `brake-disc.jpg` - для тормозного диска
- `starter.jpg` - для стартера
- `stabilizer-link.jpg` - для стойки стабилизатора

## Обновление изображений в базе данных

Если вы хотите изменить название изображения для запчасти:

```sql
UPDATE parts SET image = 'новое-название.jpg' WHERE id = 1;
```

## Placeholder

Если изображение не найдено, будет показан placeholder с иконкой.
